# 🩸 AI – Blood Cell Detection and Analysis System

**AI-powered web application** that detects and analyzes blood cells from microscopic blood smear images. The system uses **deep learning and computer vision** to automatically identify and count **Red Blood Cells (RBC), White Blood Cells (WBC), and Platelets**, helping automate part of the blood analysis process.

The application provides an **interactive web interface**, visual detection results, statistical charts, and an **automatically generated PDF report** for the analyzed image.

---

# 🚀 Features

- 🔬 Automatic detection of **RBC, WBC, and Platelets**
- 📦 Bounding box visualization for detected cells
- 📊 Real-time **cell count statistics**
- 📈 **Blood cell distribution chart**
- 📄 **Automatic PDF report generation**
- 🌐 Interactive **Streamlit web application**

# 🧪 How the System Works

1. User uploads a **blood smear image**
2. The **YOLO deep learning model** detects blood cells
3. The system identifies:
   - Red Blood Cells (RBC)
   - White Blood Cells (WBC)
   - Platelets
4. Bounding boxes are drawn around detected cells
5. The system calculates **cell counts**
6. A **distribution chart** is generated
7. A **PDF report** is automatically created for download

# 🎯 Project Objective

Manual blood cell counting is **time-consuming and prone to human error**.  
This project demonstrates how **Artificial Intelligence and Computer Vision** can automate blood cell detection to assist in medical image analysis.
